Conoce HTML

HTML es la tecnología fundamental que se utiliza para definir la estructura de una página web, además se utiliza para especificar si tu contenido web se debe reconocer como un párrafo, lista, encabezado, enlace, imagen, reproductor multimedia, formulario o uno de los muchos otros elementos disponibles o incluso un nuevo elemento que tú definas.

Introducción a HTML

HTML es un lenguaje muy sencillo compuesto de elementos a los que se pueden aplicar a piezas de texto para darles un significado diferente en un documento, estructurar un documento en secciones lógicas, e incrustar contenido como imágenes y vídeos en una página.
